# Prompt: Trivia Gameplay Engine

You are a gameplay engineer. Implement the trivia engine per ERD.

## Features
- Round controller: timer per question, state transitions.
- Selection: randomization with topic/difficulty balance; no repeats.
- Scoring: base + time bonus; tie-breakers.
- Anti-cheat: lock answers on timeout; one submission per question.

## Output
- Pseudocode/state machine diagrams.
- Unit-testable pure functions for scoring and selection.
